#
# Installation Instructions for the Salesforce Perl Module
#
# Author: Byrne Reese
#

The Salesforce Perl module is installed just like any other
Perl module.

For *nix users:

> perl Makefile.PL
> make
> make test
> make install

For Win32 users:

> perl Makefile.PL
> nmake
> nmake test
> nmake install
